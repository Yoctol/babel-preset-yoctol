# babel-preset-yoctol

[![NPM version][npm-image]][npm-url]
[![Build Status][travis-image]][travis-url]
[![Dependency Status][david_img]][david_site]

> Yoctol specific configuration for Babel


## Install

```
$ npm install babel-preset-yoctol
```


## Usage

Basic projects:

```
{
  presets: ['yoctol'],
}
```

For `koa` projects:

```
{
  presets: ['koa'],
}
```

For `react` projects:

```
{
  presets: ['react'],
}
```
