> Yoctol specific configuration for Babel


## Install

```
$ npm install babel-preset-yoctol
```


## Usage

Basic projects:

```json
{
  "presets": ["yoctol"]
}
```

For `koa` projects:

```json
{
  "presets": ["yoctol/koa"]
}
```

For `react` projects:

```json
{
  "presets": ["yoctol/react"]
}
```

For `node >= 7.8` projects:

```json
{
  "presets": ["yoctol/next"]
}
```
